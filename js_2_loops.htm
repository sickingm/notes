Javascript - loops & control structures

for loop:

for (var=startvalue;var<=endvalue;var=var+increment) 
{
    code to be executed
}

 

while loop: (only executed if condition(s) are true

while (var<=endvalue)
{
    code to be executed
}

do while loop : (always executed at least once)

do
{
    code to be executed
}
while (var<=endvalue)

switch (same as php):

switch(n)
{
case 1:
  execute code block 1
  break    
case 2:
  execute code block 2
  break
default:
  code to be executed if n is different from case 1 and 2
}

for ... in :

for (variable in object)
{
    code to be executed
}
